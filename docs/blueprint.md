## Style Guidelines:

- Primary color: Fresh blue (#22BBDB) representing clear skies and environmental focus, vibrant yet serene for data clarity.
- Background color: Very light blue-grey (#ECF6F8) providing a clean, spacious backdrop that complements the primary color and maintains readability for data visualization.
- Accent color: Verdant green (#12C474) used for highlights, positive indicators, or calls to action, creating good contrast and symbolizing health and vitality.
- Body and headline font: 'Inter' (sans-serif) for its modern, objective, and highly readable characteristics, ideal for presenting scientific data clearly and concisely.
- Use clear, concise, and modern line icons to represent navigation elements, data categories (e.g., different pollutants), and informational cues.
- Implement a modular and responsive layout with a clean dashboard-like structure. Ensure generous use of white space to enhance data legibility and avoid visual clutter across all 10 visualization pages.
- Incorporate subtle and smooth transitions for data updates on charts, filter applications, and page navigations to create a polished and dynamic user experience without distractions.
